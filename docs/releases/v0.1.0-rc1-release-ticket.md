## 风险评估
- Provider 行为差异仍可能带来边缘响应差异（由统一抽象层兜底）
- 本地网络、代理、系统权限差异可能影响 Desktop 启动稳定性
- Qdrant 不可用场景下 Memory 降级可用，但检索体验会下降

## 回滚方案
1. 回滚到上一已通过 `smoke:release` 的提交点
2. 执行：`docker compose down && docker compose up --build -d app qdrant`
3. 验证：`GET /api/health`、Web 页面可发送一轮消息
4. 若仍异常：保持回滚版本并暂停发布
