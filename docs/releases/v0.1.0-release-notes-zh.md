# v0.1.0 Release Notes（中文）

发布日期：2026-04-05  
版本：`v0.1.0`

## 本版本概览
v0.1.0 完成了本地桌面 Agent 主链路的稳定化收口，覆盖 Web 与 Desktop 启动、聊天流式链路、Tool 调用、Memory 降级与 Live2D 状态驱动，达到可持续本地运行与演示标准。

## 重点更新
- 启动链路稳定化：
  - Web/Desktop 端口冲突自动回退
  - Desktop preflight 检查纳入发布 smoke 默认流程
- 聊天主链路加固：
  - stream / stop / regenerate / fallback 一致性提升
  - requestId 幂等闭环，减少重复轮次风险
- Tool 能力闭环：
  - 手动 Tool 面板完善
  - typed params 前后端校验一致，非法参数拦截与提示可用
- Skill / MCP 体验：
  - 会话级选择记忆与恢复
  - 常见失败场景可执行提示
- Memory 闭环：
  - SQLite + Qdrant 协同可用
  - Qdrant 异常时降级不阻塞聊天主链路
- Live2D 与陪伴体验：
  - 状态驱动（idle / thinking / talking / error）
  - companion 布局与交互体验优化

## 质量与验证
本次发布前执行 `npm run smoke:release` 全量通过：
- backend tests: `42 passed`
- frontend unit tests: `77 passed`
- frontend e2e tests: `7 passed`
- startup fallback smoke: `passed`
- desktop preflight check: `passed`
- docker health check: `passed`（`/api/health` success=true）

## 已知事项
- 非阻塞项（P2）：Qdrant 不可用时仍可能出现降级 warning，但已完成降噪收口（冷却窗口 + suppressed 汇总 + 空异常文案兜底）。

## 升级与使用建议
- 建议先执行：`npm install`
- 本地联调建议优先使用：
  - `npm run local:web`
  - `npm run local:desktop`
- 发布前复验建议执行：`npm run smoke:release`
